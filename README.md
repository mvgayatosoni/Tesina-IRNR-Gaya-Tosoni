## Propósito
Este repositorio contiene los archivos utilizados para la precalibración, calibración y validación del modelo propuesto para el crecimiento poblacional de Ceratitis capitata bajo manejo con Técnica del Ínsecto Estéril. Incluye los scripts utilizados en el proceso y datos modificados que permiten utilizar y comprobar los código, y el trabajo escrito correspondiete a la tesina.
## Herramientas utilizadas
- **R** (RStudio)
- **Documentos csv** archivos que contienen los datos.
-  **Documentos PDF** trabajo escrito correspondiente a la tesina
## Colaboración con ISCAMEN
Este trabajo se realiza en colaboración con el Instituto de Sanidad y Calidad Agropecuaria de Mendoza (ISCAMEN), quien aporta los datos de campo. Al ser los datos de campo propiedad de dicha entidad, entonces los datos disponibles en este archivo estan modificados para que sean de utilidad en el uso de los códigos. No se trata de los datos reales utilizados en el trabajo.
